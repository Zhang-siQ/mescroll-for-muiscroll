# mescroll-for-muiscroll

您的项目中如果使用了mui-scroll, 则需引用本案例中的 mescroll-for-muiscroll.min.js  

<br/><br/>

mui区域滚动(mui-scroll)是通过js模拟列表滚动 -- 文档: http://dev.dcloud.net.cn/mui/ui/#scroll <br/><br/>
而 mescroll 则是原生 div 的 overflow 滚动,两者是不一样的实现方式 -- 比较: https://ask.dcloud.net.cn/article/529 <br/><br/>
如果想在 mui-scroll 中使用 mescroll 则需引用 mescroll-for-muiscroll.min.js
